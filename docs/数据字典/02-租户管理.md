# 租户管理

## 1. 租户表

**表名**: `hz_tenant`

| 序号 | 列名 | 数据类型 | 列说明 |
|------|------|---------|--------|
| 1 | tenant_id | bigint | 租户ID |
| 2 | user_id | bigint | 关联用户ID(郑好办用户) |
| 3 | tenant_name | varchar(50) | 租户姓名 |
| 4 | id_card | varchar(18) | 身份证号 |
| 5 | phone | varchar(20) | 手机号 |
| 6 | gender | char(1) | 性别(0:男 1:女) |
| 7 | birth_date | date | 出生日期 |
| 8 | education | varchar(20) | 学历 |
| 9 | marriage_status | varchar(20) | 婚姻状况 |
| 10 | work_unit | varchar(100) | 工作单位 |
| 11 | emergency_contact | varchar(50) | 紧急联系人 |
| 12 | emergency_phone | varchar(20) | 紧急联系人电话 |
| 13 | household_register | varchar(100) | 户籍地址 |
| 14 | current_address | varchar(255) | 现住址 |
| 15 | email | varchar(100) | 邮箱 |
| 16 | wechat | varchar(50) | 微信号 |
| 17 | avatar | varchar(255) | 头像URL |
| 18 | credit_score | int | 信用分(默认100) |
| 19 | is_blacklist | char(1) | 是否黑名单(0:否 1:是) |
| 20 | status | char(1) | 状态(0:正常 1:停用) |
| 21 | create_by | varchar(64) | 创建者 |
| 22 | create_time | datetime | 创建时间 |
| 23 | update_by | varchar(64) | 更新者 |
| 24 | update_time | datetime | 更新时间 |
| 25 | remark | varchar(500) | 备注 |
| 26 | del_flag | char(1) | 删除标志(0:正常 2:删除) |

## 2. 资格审核表

**表名**: `hz_qualification`

| 序号 | 列名 | 数据类型 | 列说明 |
|------|------|---------|--------|
| 1 | qualification_id | bigint | 资格ID |
| 2 | tenant_id | bigint | 租户ID |
| 3 | apply_type | char(1) | 申请类型(1:人才公寓 2:保租房 3:市场租赁) |
| 4 | apply_time | datetime | 申请时间 |
| 5 | id_card | varchar(18) | 身份证号 |
| 6 | name | varchar(50) | 姓名 |
| 7 | phone | varchar(20) | 手机号 |
| 8 | social_security_months | int | 社保缴纳月数 |
| 9 | has_local_house | char(1) | 本地是否有房(0:无 1:有) |
| 10 | education_level | varchar(20) | 学历 |
| 11 | marriage_status | varchar(20) | 婚姻状况 |
| 12 | household_count | int | 家庭人口数 |
| 13 | work_unit | varchar(100) | 工作单位 |
| 14 | monthly_income | decimal(10,2) | 月收入(元) |
| 15 | auto_check_result | char(1) | 自动审核结果(0:待审核 1:通过 2:不通过) |
| 16 | auto_check_reason | varchar(500) | 自动审核原因 |
| 17 | manual_check_result | char(1) | 人工审核结果(0:待审核 1:通过 2:不通过) |
| 18 | manual_check_reason | varchar(500) | 人工审核原因 |
| 19 | final_result | char(1) | 最终结果(0:待审核 1:通过 2:不通过) |
| 20 | check_by | varchar(64) | 审核人 |
| 21 | check_time | datetime | 审核时间 |
| 22 | status | char(1) | 状态(0:待审核 1:审核中 2:已审核) |
| 23 | create_by | varchar(64) | 创建者 |
| 24 | create_time | datetime | 创建时间 |
| 25 | update_by | varchar(64) | 更新者 |
| 26 | update_time | datetime | 更新时间 |
| 27 | remark | varchar(500) | 备注 |
| 28 | del_flag | char(1) | 删除标志(0:正常 2:删除) |
