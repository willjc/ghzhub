# 房源管理

## 1. 项目表

**表名**: `hz_project`

| 序号 | 列名 | 数据类型 | 列说明 |
|------|------|---------|--------|
| 1 | project_id | bigint | 项目ID |
| 2 | project_name | varchar(100) | 项目名称 |
| 3 | project_code | varchar(50) | 项目编码 |
| 4 | project_type | char(1) | 项目类型(1:人才公寓 2:保租房 3:市场租赁) |
| 5 | address | varchar(255) | 项目地址 |
| 6 | longitude | decimal(10,7) | 经度 |
| 7 | latitude | decimal(10,7) | 纬度 |
| 8 | total_buildings | int | 总楼栋数 |
| 9 | total_houses | int | 总房源数 |
| 10 | available_houses | int | 可用房源数 |
| 11 | project_intro | text | 项目介绍 |
| 12 | facilities | varchar(500) | 配套设施(多个用逗号分隔) |
| 13 | transportation | varchar(500) | 交通信息 |
| 14 | manager_id | bigint | 项目负责人ID |
| 15 | manager_name | varchar(50) | 项目负责人姓名 |
| 16 | manager_phone | varchar(20) | 负责人电话 |
| 17 | status | char(1) | 状态(0:正常 1:停用) |
| 18 | sort_order | int | 显示顺序 |
| 19 | create_by | varchar(64) | 创建者 |
| 20 | create_time | datetime | 创建时间 |
| 21 | update_by | varchar(64) | 更新者 |
| 22 | update_time | datetime | 更新时间 |
| 23 | remark | varchar(500) | 备注 |
| 24 | del_flag | char(1) | 删除标志(0:正常 2:删除) |
