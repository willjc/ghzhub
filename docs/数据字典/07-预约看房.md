# 预约看房

## 1. 预约看房表

**表名**: `hz_appointment`

| 序号 | 列名 | 数据类型 | 列说明 |
|------|------|---------|--------|
| 1 | appointment_id | bigint | 预约ID |
| 2 | appointment_no | varchar(50) | 预约编号 |
| 3 | tenant_id | bigint | 租户ID |
| 4 | visitor_name | varchar(50) | 看房人姓名 |
| 5 | visitor_phone | varchar(20) | 看房人电话 |
| 6 | id_card | varchar(18) | 身份证号 |
| 7 | project_id | bigint | 项目ID |
| 8 | house_id | bigint | 房源ID |
| 9 | appointment_date | date | 预约日期 |
| 10 | appointment_time | varchar(20) | 预约时间段 |
| 11 | visitor_count | int | 看房人数 |
| 12 | appointment_source | varchar(20) | 预约来源(H5/小程序/电话) |
| 13 | appointment_status | char(1) | 预约状态(0:待确认 1:已确认 2:已完成 3:已取消) |
| 14 | confirm_by | varchar(64) | 确认人 |
| 15 | confirm_time | datetime | 确认时间 |
| 16 | cancel_reason | varchar(500) | 取消原因 |
| 17 | cancel_time | datetime | 取消时间 |
| 18 | create_by | varchar(64) | 创建者 |
| 19 | create_time | datetime | 创建时间 |
| 20 | update_by | varchar(64) | 更新者 |
| 21 | update_time | datetime | 更新时间 |
| 22 | remark | varchar(500) | 备注 |
| 23 | del_flag | char(1) | 删除标志(0:正常 2:删除) |

## 2. 看房记录表

**表名**: `hz_viewing_record`

| 序号 | 列名 | 数据类型 | 列说明 |
|------|------|---------|--------|
| 1 | record_id | bigint | 记录ID |
| 2 | appointment_id | bigint | 预约ID |
| 3 | tenant_id | bigint | 租户ID |
| 4 | visitor_name | varchar(50) | 看房人姓名 |
| 5 | project_id | bigint | 项目ID |
| 6 | house_id | bigint | 房源ID |
| 7 | viewing_date | date | 看房日期 |
| 8 | viewing_time | datetime | 看房时间 |
| 9 | guide_person | varchar(50) | 带看人 |
| 10 | viewing_duration | int | 看房时长(分钟) |
| 11 | satisfaction_score | int | 满意度评分(1-5) |
| 12 | is_interested | char(1) | 是否有意向(0:否 1:是) |
| 13 | feedback | varchar(500) | 反馈意见 |
| 14 | create_by | varchar(64) | 创建者 |
| 15 | create_time | datetime | 创建时间 |
| 16 | update_by | varchar(64) | 更新者 |
| 17 | update_time | datetime | 更新时间 |
| 18 | remark | varchar(500) | 备注 |
| 19 | del_flag | char(1) | 删除标志(0:正常 2:删除) |
