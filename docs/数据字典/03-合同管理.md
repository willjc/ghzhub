# 合同管理

## 1. 合同表

**表名**: `hz_contract`

| 序号 | 列名 | 数据类型 | 列说明 |
|------|------|---------|--------|
| 1 | contract_id | bigint | 合同ID |
| 2 | contract_no | varchar(50) | 合同编号 |
| 3 | contract_type | char(1) | 合同类型(1:人才公寓 2:保租房 3:市场租赁) |
| 4 | template_id | bigint | 合同模板ID |
| 5 | tenant_id | bigint | 租户ID |
| 6 | tenant_name | varchar(50) | 租户姓名 |
| 7 | tenant_id_card | varchar(18) | 租户身份证 |
| 8 | tenant_phone | varchar(20) | 租户手机 |
| 9 | project_id | bigint | 项目ID |
| 10 | house_id | bigint | 房源ID |
| 11 | house_code | varchar(50) | 房源编码 |
| 12 | house_address | varchar(255) | 房源地址 |
| 13 | rent_price | decimal(10,2) | 租金(元/月) |
| 14 | deposit | decimal(10,2) | 押金(元) |
| 15 | start_date | date | 租赁开始日期 |
| 16 | end_date | date | 租赁结束日期 |
| 17 | rent_months | int | 租赁月数 |
| 18 | payment_cycle | char(1) | 缴费周期(1:月付 2:季付 3:半年付 4:年付) |
| 19 | payment_day | int | 每月缴费日 |
| 20 | contract_content | text | 合同内容 |
| 21 | contract_file | varchar(255) | 合同文件URL |
| 22 | tenant_signature | text | 租户签名数据(base64) |
| 23 | landlord_signature | text | 房东签名数据(base64) |
| 24 | sign_time | datetime | 签署时间 |
| 25 | contract_status | char(1) | 合同状态(0:草稿 1:待签署 2:已签署 3:履行中 4:已到期 5:已解约) |
| 26 | is_renewed | char(1) | 是否续约(0:否 1:是) |
| 27 | renewed_contract_id | bigint | 续约合同ID |
| 28 | create_by | varchar(64) | 创建者 |
| 29 | create_time | datetime | 创建时间 |
| 30 | update_by | varchar(64) | 更新者 |
| 31 | update_time | datetime | 更新时间 |
| 32 | remark | varchar(500) | 备注 |
| 33 | del_flag | char(1) | 删除标志(0:正常 2:删除) |
