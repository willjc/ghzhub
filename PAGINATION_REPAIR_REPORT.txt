【根本问题】
================================================================================

错误的分页模式（当前多个Controller存在）:
┌─────────────────────────────────────────────────────────────────┐
│ startPage();  // ThreadLocal存储分页参数，但接下来的查询未使用  │
│ List<T> list = service.selectXxxList(xxx);  // 返回全量List     │
│ return getDataTable(list);  // total = list.size() 分页无效     │
└─────────────────────────────────────────────────────────────────┘

正确的分页模式（已修复4个Controller采用）:
┌─────────────────────────────────────────────────────────────────┐
│ Page<T> page = PageUtils.getPage();                             │
│ IPage<T> result = service.selectXxxPage(xxx, page.getCurrent(), │
│     page.getSize());  // MyBatis-Plus处理分页                   │
│ TableDataInfo.setRows(result.getRecords());  // 当前页数据      │
│ TableDataInfo.setTotal(result.getTotal());   // 正确的总数      │
└─────────────────────────────────────────────────────────────────┘
