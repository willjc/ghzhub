# 港好住平台 - 老数据准备清单

> 请按照此清单准备需要迁移的老系统数据

---

## 一、基础房源数据 (最优先)

| 表名 | 说明 | 需要准备的内容 |
|------|------|---------------|
| **hz_project** | 项目信息 | 项目名称、编码、类型、地址、经纬度、封面图、租金详情、物业信息、项目简介 |
| **hz_building** | 楼栋信息 | 楼栋名称、编码、所属项目、总楼层、总单元数 |
| **hz_unit** | 单元信息 | 单元名称、所属楼栋 |
| **hz_house_type** | 户型信息 | 户型名称、户型描述、户型图片 |
| **hz_house** | 房源信息 | 房源编码、房号、楼层、户型、面积、租金、押金、朝向、装修、房源状态 |
| **hz_house_image** | 房源图片 | 房屋图片路径 |
| **hz_house_vr** | VR看房 | VR链接 |

---

## 二、客户/租户数据

| 表名 | 说明 | 需要准备的内容 |
|------|------|---------------|
| **hz_tenant** | 租户信息 | 姓名、身份证号、手机号、性别、出生日期、学历、工作单位、紧急联系人、户口地址、现住址 |
| **hz_user** | 用户账号 | 用户名、密码、昵称、头像 |
| **hz_enterprise** | 企业信息 | 企业名称、统一社会信用代码、联系人、联系电话、企业地址 |
| **hz_co_tenant** | 合租人信息 | 合租人姓名、身份证号、手机号 |
| **hz_tenant_document** | 租户资料 | 身份证、户口本、学历证、工作证明等材料 |

---

## 三、合同数据

| 表名 | 说明 | 需要准备的内容 |
|------|------|---------------|
| **hz_contract** | 租赁合同 | 合同编号、租户信息、房源信息、租金、押金、租期起止日期、付款周期、签署状态 |
| **hz_contract_template** | 合同模板 | 模板名称、模板内容 |
| **hz_contract_attachment** | 合同附件 | 附件文件路径 |

---

## 四、入住/退租数据

| 表名 | 说明 | 需要准备的内容 |
|------|------|---------------|
| **hz_checkin_record** | 入住记录 | 租户、房源、入住时间、经办人 |
| **hz_checkout_record** | 退租记录 | 租户、房源、退租时间、退租原因、费用结算 |
| **hz_qualification** | 资格校验 | 租户信息、校验结果、校验材料 |
| **hz_checkin_apply** | 入住申请 | 申请记录、审核状态 |
| **hz_checkout_apply** | 退租申请 | 申请记录、审核状态 |
| **hz_qualification_appeal** | 资格申诉 | 申诉原因、申诉材料 |

---

## 五、财务账单数据

| 表名 | 说明 | 需要准备的内容 |
|------|------|---------------|
| **hz_bill** | 账单信息 | 账单类型(租金/押金/物业费/水电费)、金额、账单周期、缴费状态 |
| **hz_payment** | 缴费记录 | 缴费金额、缴费时间、缴费方式 |
| **hz_refund_apply** | 退款申请 | 退款金额、退款原因、退款状态 |
| **hz_refund_record** | 退款记录 | 实际退款记录 |
| **hz_invoice_apply** | 发票申请 | 发票抬头、税号、开票金额、开票状态 |

## 七、不需要准备的表

以下表为系统自动生成或框架自带，**不需要准备数据**：

| 表类型 | 说明 |
|--------|------|
| `sys_*` | 若依框架系统管理表 (用户、角色、菜单、字典等) |
| `qrtz_*` | 定时任务框架表 |
| `gen_*` | 代码生成器表 |
| `hz_item_status` | 项目状态 (系统自动维护) |
| `hz_inventory_list` | 库存清单 (系统自动统计) |
| `hz_operation_config` | 运营配置 (系统配置) |
